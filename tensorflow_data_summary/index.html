TensorFlow数据总结

发表于 2017-09-05 | 分类于 TensorFlow | 阅读次数:

TensorFlow中的数据叫做Tensor,中文名叫做张量。由于张量要表示任意类型的数据，如照片、语音等等，那么普通的简单数据类型如整型、浮点数等一般很难满足这个需求。自然而然的，就想到了组合型数据类型，如一维数组、多维数组、哈希表、字符串。理论上，这些组合型数据都可以满足表示任何数据的要求，唯一的不同只是谁更适合而以。TensorFlow其内部运行机制（要进行大量矩阵运算）决定了它的数据类型为多维数组。多维数组与矩阵天然的相似性，为计算提供了诸多方便。
下面分别介绍TensorFlow常用的几种数据类型：常量、变量、占位变量。它们都不是单个的数据，如一个整数，而是指多维数组，如数据类型为整数的2行2列的多维数组。

TensorFlow 常量

常量就是不可更改的数据。要注意，这里的数据不是单个的数据，是指多维数组。其初始化方法如下。

参数:
value: 一个数据类型为dtype的常数值或者常数列表。
dtype: 常量的数据类型（可选）。
shape: 常量的维度（可选）。
name: 张量的名字（可选）。
verify_shape: 张量的形状是否可更改。
